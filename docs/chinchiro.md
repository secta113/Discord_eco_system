# チンチロリン (Chinchiro) 詳細仕様

> **最終更新**: 2026-03-30  
> *v2.11 ジャックポット共通システムへの移行に基づき更新*

---

## 1. 基本ルール

- **概要**: 3つのダイスを振り、出た目の組み合わせ（役）の強さを競います。
- **試行回数**: 各プレイヤーは1ターンに最大 **3回** までダイスを振ることができます。
- **役の確定**:
  - 役（目なし以外）が出た時点でそのターンの役が確定し、次のプレイヤーへ交代します。
  - 3回振っても役が出なかった場合は「目なし」として確定します。
- **勝敗判定**: 全員の役が確定した後、最も「強さ（Strength）」が高いプレイヤーがポットの全額を獲得します。

---

## 2. 役一覧と強さ (Hand & Strength)

| 役名 | 組み合わせ | 強さ (Strength) | レアリティ | ジャックポット配当 | キャップ |
| :--- | :--- | :--- | :--- | :--- | :--- |
| **ピンゾロ** | `1-1-1` | **5000** | RARE | **10%** 分配 | 10,000 pts |
| **アラシ** | `n-n-n` (n=2~6) | **2000 + n** | COMMON | **3%** 分配 | 10,000 pts |
| **シゴロ** | `4-5-6` | **1000** | - | - | - |
| **○の目** | `x-x-p` | **100 + p** | - | - | - |
| **目なし** | 上記以外 | **0** | - | - | - |
| **ヒフミ** | `1-2-3` | **-1000** | - | - | - |

## 3. 特別な清算ルール

### A. 全員目なし (All No-Role)
- 全参加者の役が「目なし（またはヒフミ）」であった場合、そのゲームのポット全額は誰にも支払われず、**ジャックポットプール（システムアカウント ID:0）へ没収** されます。

### B. 親の責任 (Oya Tax)
- **条件**: 2人以上の対戦（PvP）において、**募集主（リストの先頭）** が勝利した場合。
- **内容**: 獲得賞金の **5%** がシステム手数料として徴収され、ジャックポットプールへ蓄積されます。

---

## 4. 接待アルゴリズム (Hospitality)

プレイヤーの不快感を軽減するための救済ロジックです。
- **発動**: 3回目のロールで「目なし」が確定した場合に一定確率で介入。
- **JPペナルティ**: 接待発動時は、その回のジャックポット当選権利が **無効化** されます。
- 詳細は [接待ロジック詳細仕様 (hospitality_logic.md)](file:///d:/python_project/Discord_eco_system/docs/hospitality_logic.md) を参照してください。
