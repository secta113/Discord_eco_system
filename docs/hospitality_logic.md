# 接待ロジック詳細仕様 (hospitality_logic.md)

チンチロリンおよびブラックジャックに実装されている「接待（Bad Luck Protection）」システムの内部仕様です。

> [!NOTE]
> **実装状況**: 本システムは v2.4 仕様に基づき、ブラックジャックのステータス名不整合修正およびチンチロリンのタイミング適正化が完了しています。

## 1. 接待レート (v2.4)
介入が発生する基本確率は、ユーザーの資産ステータスによって決まります。

| ステータス | 接待率 (Hospitality Rate) | 判定条件 |
| :--- | :--- | :--- |
| **Prime** | **5%** | 残高 > Benchmark |
| **Standard** | **20%** | Benchmark * 0.3 ≤ 残高 ≤ Benchmark |
| **Recovery** | **40%** | 残高 < Benchmark * 0.3 |

---

## 2. ゲーム別の介入内容

### チンチロリン (Chinchiro)
ロール（振る）フェーズで「目なし」を回避するために介入が発生します。

- **発動条件**: 3回目のロールで「目なし（役なし）」が確定した場合。
- **処理内容**:
    1.  ユーザーの「接待率」に基づきランダム判定。
    2.  成功した場合、内部的に**リロール（振り直し）**を実行。
    3.  リロール後も「目なし」だった場合、不快感回避のため**ランダムな「○の目（1〜6の目）」を強制的に付与**します。
- **演出**: メッセージ `✨ 幸運の女神が微笑んでいるようです！` が表示されます。

### ブラックジャック (Blackjack)
プレイヤーのバースト回避と、ディーラーのバースト誘導の2箇所で介入が発生します。

#### A. プレイヤーのバースト回避 (Hit時)
- **発動条件**: プレイヤーの手札が **12〜16** の状態で「Hit」を選択し、次に山札から引くカードが **10相当のカード**（バースト確定）である場合。
- **処理内容**:
    - ユーザーの「接待率」に基づきランダム判定。
    - 成功した場合、その10相当のカードを **山札の一番下へ送り**、別のカードを引かせます。
    - **【重要：適用範囲】** 接待によるペナルティ（希少役無効化）は、**接待が発生した手札のみ**に適用されます（スプリットした別の手札には影響しません）。
    - ※回避後に別の札でバーストした場合は、不自然さを避けるため「幸運の女神〜」の演出は表示されません。

#### B. ディーラーのバースト誘導 (ディーラー手番)
- **発動条件**: ディーラーの手札が **12〜16** で、次のカードを引くと **17〜21** （バーストしない）に収まる場合。
- **処理内容**:
    - 参加者に `Standard` または `Recovery` がいる場合。
    - **【重要：例外規定】** プレイヤーの中に一人でも**特殊役（777, 678, 6-Card Charlie, Suited BJ）が成立している者がいる場合、そのプレイヤーのジャックポット権利を保護するため、ディーラーの接待介入はスキップされます。**
    - 確率は Standard: **20%**、Recovery: **40%**。
    - 成功した場合、山札から **10相当のカードを検索して差し替え**、ディーラーを強制的にバーストさせます。
    - **【重要：適用範囲】** ディーラーが強制バーストした場合、その恩恵を受けるプレイヤーの**全手札（スプリットを含む）**にペナルティが適用されます。

---

## 3. 女神のペナルティ (アビューズ対策)
接待ロジックによって救済された回で、偶然に「希少役（ジャックポット対象）」が完成した場合、その配当は無効化されます。

- **表示メッセージ**: `⚠️ 女神がボーナスを持ち逃げしました…`
- **対象外となる役**:
    - **チンチロリン**: ピンゾロ、アラシ
    - **ブラックジャック**: 7-7-7、6-7-8、6-Card Charlie、Suited BJ
