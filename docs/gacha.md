## 1. コンセプト：結果ではなく「出来事」を集める

本システムは、ポイントを増やすこと自体を主目的とするのではなく、**「システム内に用意された全250種類のどうでもいい日常イベントをコンプリートすること」** をユーザーのモチベーションとする。

すべての結果にクスッとくる（あるいはイラッとする）くだらないフレーバーテキスト（物語）を付与することで、単なるアタリ・ハズレではなく「出来事」として体験させる。

### 導入する心理的フック

- **大規模コレクション要素**: 全250種のうち「まだ見ていないものを引きたい」という収集欲求。
- **高コスト・高ボラティリティ**: 参加費を高く設定し、引く際の「ヒリヒリ感」を演出。
- **究極のアンチカタルシス**: 0.4%で引ける最高レアリティが「配当0pt」という逆転の発想。

## 2. 実行ルール（v1.6: 1日3回・コスト増分制）

| 実行回数（当日） | コスト | 特典・制約 |
| :--- | :--- | :--- |
| **1回目** | **500 pt** | **【新規確定】** 未所持のイベントが必ず出現 |
| **2回目** | **1,000 pt** | 【ランダム】重複（所持済み）の可能性あり |
| **3回目** | **1,500 pt** | 【ランダム】重複の可能性あり |
| **4回目以降** | **不可** | 翌日 JST 0:00 のリセットまで実行できない |

**収支バランス（期待値: 1,072 pt）**:
- 1回目（コスト 500 pt）: 平均 **+572 pt**（大幅増・救済）
- 2回目（コスト 1,000 pt）: 平均 **+72 pt**（微増）
- 3回目（コスト 1,500 pt）: 平均 **−428 pt**（収集コスト）

## 3. 確率テーブルと配当設計

抽選は `gacha_event.json` の全250件を `random.choice()` で**等確率**に選出します。  
排出率は JSON 内のレアリティ別件数の比率で決まります（件数ベース確率）。

| カテゴリ | JSON件数 | 実排出率 | 払出額 | 期待値寄与 |
| :--- | :--- | :--- | :--- | :--- |
| 🌈 **Ultimate（究極の幸運）** | 1件 | **0.4%** | **0 pt** | 0 pt |
| 🟢 **Legendary（伝説級の幸運）** | 3件 | **1.2%** | **15,000 pt** | 180 pt |
| 🔵 **Epic（大成功・大発見）** | 10件 | **4.0%** | **4,000 pt** | 160 pt |
| 🟡 **Rare（ちょっとした幸運）** | 36件 | **14.4%** | **2,000 pt** | 288 pt |
| ⚪ **Normal（平穏な日常）** | 88件 | **35.2%** | **1,000 pt** | 352 pt |
| 🟠 **Bad（不運・トラブル）** | 75件 | **30.0%** | **400 pt** | 120 pt |
| 🔴 **Disaster（致命的な大失敗）** | 37件 | **14.8%** | **0 pt** | 0 pt |
| **合計** | **250件** | **100%** | — | **約1,100 pt** |

> [!NOTE]
> 配当・期待値を調整するには `gacha_event.json` の各レアリティの件数比率か `PAYOUT_MAP` を変更してください。

## 4. ロジック仕様

1. **回数判定**: `gacha_count_today` が 3 未満であることを確認する。
2. **新規確定モード**: `gacha_count_today == 0` の場合、全250件から**未所持IDのイベント**を候補リストに絞り込み、`random.choice()` で等確率に1件選出する。
3. **ランダムモード**: 2回目以降、または全種コンプリート済みの場合、重複を許可し全250件から `random.choice()` で選出する。
4. **日付リセット**: `last_gacha_daily` が今日（JST）でない場合、`gacha_count_today` を 0 にリセットする。
5. **配当マップ**: 選出したイベントの `rarity` フィールドを `PAYOUT_MAP`（コード定数）に照合し、払出額を決定する。

## 5. フレーバーテキスト例

### 🌈 Ultimate（配当 0 pt / 名誉のみ）
- 「空から降ってきた鉄骨が頭上数ミリをかすめた。無傷で済んだことで一生分の運をすべて使い果たしたため、もうポイントは必要なくなった。」
- 「落とした100円玉が転がってマンホールの隙間に向かい……落ちる直前でピタリと縦に立った。この宇宙の真理に触れたため、ポイントは没収された。」

### 🟢 Legendary（15,000 pt）
- 「ピノを開けたら、星型とハート型が両方入っていた。今年の運をすべて使い果たした気がする。」
- 「道に落ちていた宝くじを交番に届けたら、後日持ち主が現れず自分のものになり、しかも3万円当たっていた。」

### 🔵 Epic（4,000 pt）
- 「目覚ましが鳴る1分前に自然と目が覚めた。完璧な目覚めに勝利を確信し、そのまま二度寝した。」
- 「コンビニのレジで会計がピッタリ『777円』だった。店員と謎の連帯感が生まれた。」

### 🟡 Rare（2,000 pt）
- 「USBケーブルをPCに挿そうとしたら、1回目のチャレンジで表裏がピタリと合って挿さった。」
- 「信号が連続で青だった。まるで世界が自分を歓迎しているかのようにスムーズに帰宅できた。」

### ⚪ Normal（1,000 pt）
- 「道端で猫を見かけたので『チッチッチ』と呼んでみたが、完全に無視された。」
- 「天気予報で降水確率0%だったので傘を持たずに出かけたら、普通に晴れた。予報通りだった。」

### 🟠 Bad（400 pt）
- 「新しい靴下を履いた直後に、キッチンの床にこぼれた謎の水滴を踏んでしまった。」
- 「カップ麺にお湯を注いだ後、3分待たずに寝落ちし、目が覚めたらスープを全吸収したオブジェが完成していた。」

### 🔴 Disaster（0 pt）
- 「イヤホンがBluetoothに繋がっていると思い込み、電車内で大音量のアニソンをスピーカーから垂れ流した。」
- 「ズボンのチャック全開で半日過ごしていたことに、帰宅して鏡を見た瞬間に気づいた。今日の出会いすべてを消去したい。」
