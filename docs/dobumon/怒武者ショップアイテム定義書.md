# **v1.4 怒武者（ドブモン）ショップアイテム定義書**

本ドキュメントは、怒武者の内部パラメータ（IV、禁忌、収穫逓減）に干渉するためのアイテムリストである。

**【重要】インベントリ機能が存在しないため、購入と同時に効果が発動・適用される「即時消費型」として設計されている。**

## **0\. 消費仕様（即時発動ルール）**

1. **ステータス操作系**: 購入時に選択していた怒武者に即座に反映される。  
2. **次動予約系**: 交配用アイテムや戦闘用バフなどは、購入したユーザーに「予約フラグ」を付与し、次に行われる該当アクションで自動消費される。  
3. **回数・期間系**: 購入時から一定回数、または一定期間（当日中など）効果が持続する。

## **1\. 闇の露店**

**コンセプト**: 禁忌深度を代償に、あるいは禁忌そのものを利用した「理外」の成長・能力。

| アイテム名 | 効果説明（購入時即発動） | 数値的裏付け |
| :---- | :---- | :---- |
| **生贄の刻印** | **【次動予約】** 次に行う交配で、子の**禁忌深度を強制 \+1** する代わり、全IV期待値を 0.2 底上げ。 | next\_breed\_iv\_bonus \+= 0.2, next\_taboo\_add \+= 1 |
| **忌血の触媒** | **【次動予約】** 次に行う戦闘中のみ、自身の**禁忌深度に応じて**攻撃力を倍加。 | next\_battle\_atk\_buff \= 1.0 \+ (depth \* 0.4) |
| **自滅の劇薬** | **【即時反映】** 選択中の個体の寿命を 15 減らし、**次の一回**のトレーニング上昇量を 3.0倍 にする。 | training\_multiplier \= 3.0, lifespan \-= 15 |

## **2\. ドブ屋**

**コンセプト**: 劣悪な品質による「揺らぎ」と副作用。リスクを許容する層向け。

| アイテム名 | 効果説明（購入時即発動） | 数値的裏付け |
| :---- | :---- | :---- |
| **腐ったプロテイン** | **【即時反映】** **次の一回**のトレーニング上昇量を 0.1 〜 2.0倍 の間でランダム化。 | training\_multiplier \= random(0.1, 2.0) |
| **重すぎる鉄下駄** | **【状態付与】** 選択中の個体に「鈍足」状態を付与。SPD成長が半減するが、ATK上昇効率が 1.2倍 になる。 | SPD\_growth\_debt \= 0.5, ATK\_growth\_buff \= 1.2 |
| **使い古しの参考書** | **【即時反映】** 選択中の個体が、ランダムなデフォルト技を1つ習得。 | learn\_random\_skill(selected\_dobumon) |

## **3\. 町の雑貨屋**

**コンセプト**: 絆（Affection）の向上と、確実な育成支援。

| アイテム名 | 効果説明（購入時即発動） | 数値的裏付け |
| :---- | :---- | :---- |
| **高級菓子折り** | **【即時反映】** 選択中の個体との絆（Affection）を直接 **\+10** 上昇。 | affection \+= 10 |
| **誓いの契約書** | **【次動予約】** 次に行う交配で、選択した親の「特定の技」を **100% 継承** させる。 | next\_breed\_inherit\_fixed \= True |
| **無地の巻物** | **【即時反映】** 選択中の個体の技の命名権を再取得。※一度も命名していない場合に限る。 | can\_rename\_skill \= True |

## **4\. マッスルランデブー**

**コンセプト**: 成長関数（収穫逓減）と過労状態へのメタ介入。

| アイテム名 | 効果説明（購入時即発動） | 数値的裏付け |
| :---- | :---- | :---- |
| **筋繊維ブースター** | **【状態付与】** 以降 5 回分のトレーニング上昇量を **1.1倍**（整腸効果）にする。 | training\_multiplier\_stack \= 5, multiplier \= 1.1 |
| **冷却シップ** | **【即時反映】** 本日のトレーニング回数（today\_train\_count）を **\-2** する（費用リセット目的）。 | today\_train\_count \= max(0, \-2) |
| **超回復サプリ** | **【即時反映】** **次の一回**のみ、収穫逓減（Diminishing Returns）による減衰計算を無効化。 | ignore\_diminishing\_returns \= True |

## **5\. Gengularity（ゲンギュラリティ）**

**コンセプト**: 遺伝子レベルの改竄、IVの限界突破。

| アイテム名 | 効果説明（購入時即発動） | 数値的裏付け |
| :---- | :---- | :---- |
| **特異点の断片** | **【即時反映】** 指定したステータスのIVを直接 **\+0.02** 加算（極致への介入）。 | stat\_iv \+= 0.02 |
| **ミューテーション・ゲノム** | **【次動予約】** 次に行う交配で、子の突然変異率を **\+20%** 加算。 | next\_mutation\_rate \+= 0.20 |
| **理の崩壊** | **【次動予約】** 次に行う交配で、潜在能力（potential）を強制的に「💥 爆発（burst）」へ書き換える。 | next\_potential\_gene \= "burst" |

## **💡 システム・フラグ管理の補足**

* **フラグの重複**: 同系統の「次動予約」アイテムを連続購入した場合、効果は上書きされるのか、加算されるのか（例：ミューテーション・ゲノム×2）は、実装時のバランス調整に依存。  
* **対象選択**: ショップメニューの決定（ステップ4）の際、どのドブモンに適用するかを選択させるUIフローが必要。
