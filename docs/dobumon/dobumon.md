## 1. システム概要
「怒武者（ドブモン）」は、Discordボット「Dealer-bot」における新たな経済圏および対戦コンテンツとして定義される、ハイリスク・ハイリターンな育成シミュレーションゲームです。
ギャンブルで得たポイントを消費してモンスターを育成し、プレイヤー同士が文字通り「命（データ）を懸けて」戦うことで、スリリングな体験とポイントのインフレ抑制（デフレ対策）を両立させています。

## 2. 内部構造（アーキテクチャ）
システムは以下のモジュールに分離され、高い保守性と拡張性を備えています。

- **`DobumonCog`**: コマンド受付とUI起動。
- **`DobumonService`**: 育成・決闘・交配などのビジネスロジック。
- **`DobumonManager`**: データ永続化とステータス計算。
- **`DobumonFormatter`**: ANSI/Markdown を用いたプレミアムUI整形。
- **`BattleEngine`**: 戦闘シミュレーション計算。
- **`TrainingEngine`**: 収穫逓減ロジックを含む育成計算。
- **`BreedingFactory`**: 遺伝情報の継承ロジック。内部で `MutationEngine` を利用。
- **`MutationEngine`**: 突然変異の発生と遺伝子座の固定（ホモ接合化）を一元管理。

## 3. 個体設計（ステータス）
すべてのステータスは内部的に **`float` (浮動小数点)** で管理され、微細な個体値の差や成長の蓄積を正確に保存します。表示の際は視認性を優先して整数に丸められます。

### 3.1 公開パラメータ
| パラメータ | 説明 |
| :--- | :--- |
| **HP / health** | 生命力。戦闘後は `health` に残りHPが保存される。0になると死亡。 |
| **ATK / DEF** | 攻撃力と防御力。ダメージ計算の基準となる。 |
| **EVA / SPD** | 回避力と行動力。回避率と先制権、手数に影響。 |
| **絆 (Affection)** | 懐き度。トレーニングで上昇し、セリフや技の効果に影響（予定）。 |
| **属性** | 火・水・草の3すくみ関係。弱点倍率による有利不利を決定。 |
| **性別** | オス(M)・メス(F)。交配の組み合わせに影響。 |

### 3.2 隠しパラメータ
| パラメータ | 説明 |
| :--- | :--- |
| **IV (個体値)** | ステータスごとの補正係数。取得元により異なる：バイヤー=`0.9〜1.1`、ガチャ=`1.0〜1.4`。交配では親の平均をベースに変動し、配偶ボーナスの積み重ね（IV Drift）により **2.0** を超える「極致」に達する場合がある。 |
| **寿命** | 最大生存期間（トレーニング可能回数）。禁忌深度により指数関数的に減少する。 |
| **病の発症率** | 突然のデバフや死亡のリスク。近親交配 (COI) や禁忌深度により上昇する。 |
| **遺伝子** | メンデルの法則に従い、両親から1つずつアレルを継承する。 |

## 4. 獲得と育成
### 4.1 獲得方法
1. **バイヤー**: 定額ポイントで購入（標準個体）。
2. **ガチャ**: 既存のガチャ機能から低確率で排出（高ステータス個体）。
3. **高額配当報酬**: ギャンブルで一定額以上の配当を得た際、その額に応じた初期値を持つ怒武者を獲得。
4. **交配**: 所持している怒武者同士を掛け合わせて新個体を誕生させる。

### 4.2 育成（Training Engine v2.x）
- **成長関数**: `基本上昇量 × IV × 乱数(0.8~1.2)` で算出。
- **大成功**: 一定確率で上昇量が 1.5 倍になる。
- **収穫逓減 (Diminishing Returns)**: ステータスが高いほど上昇効率が低下し、無限のインフレを抑制。
- **オーバーワーク**: 1日に5回を超える（6回目以降、内部: `today_train_count >= 5`）と寿命が削れるリスク（病気）が発生。特訓開始時に警告が表示されます（※マッサージは免除）。
- **トレーニング費用**: 1日の回数に応じて増加するステップアップ制（500 -> 1000 -> 2000 -> 4000 pts）。
- **懐き度割引**: なつき度（Affection）に応じて最大 **50%** の費用割引が適用されます。

### 4.3 技の習得と命名
ステータスの閾値到達や、トレーニング中のランダムイベントにより「技」を習得します（最大4つ）。
習得した技（デフォルト名）は、一度だけユーザーが自由に命名（`/dd-dobumon rename-skill`）できます。
**⚠️ 重要: 命名後の変更は一切不可です。**

### 4.4 絆（KIZUNA / Affection）
懐き度の数値に応じた叙情的なメッセージが表示されます（7段階）。
- `🥀 絆 ( < 0 )`: あなたの視線を避け、どこか悲しげにそっぽを向いている。
- `🍂 絆 ( < 5 )`: 呼んでも無視されてしまう。まだ信頼関係は築けていない。
- `🌱 絆 ( < 15 )`: たまにこちらをチラチラ見てくれるが、まだ距離を感じる。
- `💖 絆 ( < 30 )`: あなたの存在に少しずつ慣れ、信頼が芽生え始めている。
- `✨ 絆 ( < 50 )`: 手が近づくと、嬉しそうに目を細めるようになった。
- `🤝 絆 ( < 70 )`: 唯一無二のパートナーとして、固い絆で結ばれている。
- `🔥 絆 ( < 100 )`: 言葉を交わさずとも心が通じ合っている。
- `💎 絆 ( 100+ )`: 不滅の愛と信頼を、あなたという存在にすべて捧げている。

#### **一日の上昇制限**
- トレーニングによるなつき度の上昇は **1日最大 8 ポイント** までです。
- **安全圏外（オーバーワーク状態）** でのトレーニングではなつき度は上昇しません。
- マッサージは1日1回目のみ **+2** 加算されます（2回目以降は回復のみ）。
- 通常トレーニングは **+1**、大成功時は **+2** 加算されます。

## 5. 遺伝・血統システム (Genetics)

強さと個性を決定づける遺伝システムおよび血統理論についての詳細です。

### 5.1 継承の基本アルゴリズム
怒武者の誕生時、以下の計算式に基づいて能力が決定されます。

- **ステータス実数値**: `(両親の平均値 × 子供のIV) ＋ 遺伝努力値(0〜5)` 。これに特性の最終倍率が乗る。
- **個体値 (IV)**: `(両親のIV平均値) × 変動係数(0.9〜1.4) × 配偶ボーナス`。配偶ボーナスにより期待値が 1.0 を超えるため、世代を重ねるほど血統の底上げ（**IV Drift**）が発生する。
- **スキル**: 両親が持つユニークな技の中から、1〜2個をランダムに引き継ぐ。
- **属性**: 火・水・草の属性は親に関わらず、誕生時にランダムに再抽選される。

### 5.2 メンデリズム (Mendelian Genetics)
怒武者には4つの主要な遺伝子座（スロット）があり、それぞれ「優性(D)」と「劣性(r)」のアレルペアによって表現型が決まります。

| 遺伝子座 | 優性(D) | 劣性(r) | 説明 |
| :--- | :--- | :--- | :--- |
| **成長型 (`growth`)** | 🐣 早熟 (`early`) | 🌙 晩成 (`late`) | 成長曲線と初期値に影響 |
| **生命力 (`vitality`)** | 💎 金剛 (`hardy`) | 🍃 繊細 (`frail`) | 病気率と寿命に影響 |
| **潜在能力 (`potential`)** | ⚖️ 安定 (`stable`) | 💥 爆発 (`burst`) | IVの揺らぎと突然変異率に影響 |
| **肉体内包 (`body`)** | ✨ 通常 (`normal`) | ✨ 美形 (`aesthetic`) | 外見的特徴と回避性能に影響 |

- **優劣の法則**: スロットに1つでも「D」があれば優性形質が発現し、両方が「r」の場合のみ劣性形質が発現します。

### 5.3 血液の不均衡と「禁忌」のしこり
オス（M）は「赤の血（濃すぎる血）」、メス（F）は「青の血（薄すぎる血）」を持ちます。特定の組み合わせは「禁忌」を招きます。

[//]: # ("禁忌"は"禁忌"以外の単語では呼称しません。もし、"禁断"の呼称が使用された場合は、"禁忌"に置換して回答し、"禁忌"以外の単語は使用しないこと。)

| 配合タイプ | 発生因子 | 遺伝ボーナス | 特徴 |
| :--- | :--- | :--- | :--- |
| **通常配偶 (M+F)** | なし | 標準的な継承 | 最も安定した血統。 |
| **百合配偶 (F+F)** | **🔵 青の禁忌** | **IV x1.15** | 高い素質を引き継ぐ。**寿命減衰**が激しい。 |
| **薔薇配偶 (M+M)** | **🔴 赤の禁忌** | **IV x1.20 ＆ 全ステータス x1.20** | 圧倒的性能。不妊かつ病弱。 |

- **🔴 赤の禁忌 (MM)**: 攻撃力(1.5x)・速度(1.2x)。代償：**不妊**、**病弱 (+15%)**、延命不可。
- **🔵 青の禁忌 (FF)**: 回避(1.2x)・HP(1.2x)。代償：**寿命減衰 (x0.5)**、延命不可。
- **血のしこり（禁忌深度）**: 禁忌因子は通常の交配に戻しても消えず、代々そのリスクを積層（スタック）させます。
    - **初期寿命の崩壊**: `誕生時の寿命 = ベース寿命(80~120) × 寿命補正係数 ^ 禁忌深度`
    - **老化の加速**: `寿命消費倍率 = 通常消費 × 1.2 ^ 禁忌深度`
    - **対禁忌への脆弱性**: 特性「対禁忌」を持つ相手に対し、相手の攻撃・防御を `1.0 + (深度 × 0.3)` 倍に強化させてしまう。

### 5.4 突然変異と希少形質
ブリーディング時に極低確率（1%〜）で発生する希少な変異体です。これらは通常の遺伝とは別に、各遺伝子座（スロット）が「概念的な変貌」を遂げることで発現します。

#### **🧬 肉体内包 (`body`) の変異**
- **👑 金角 (`gold_horn`)**: 威厳ある角。防御力が **1.2倍** に向上し、勝利時の獲得報酬が **1.1倍** に増加する。
- **🎒 赤背 (`red_back`)**: 激しい闘争本能。攻撃力が **1.3倍** になる。
- **👁️ 妖眼 (`odd_eye`)**: 異なる次元を視る瞳。回避力が **1.3倍** に向上する。
- **💉 青血 (`blue_blood`)**: 高貴なエイリアン・ブラッド。HPが **1.3倍** になり、病気にかかる確率が **半減（0.5倍）** する。
- **🔪 硝子の刃 (`glass_blade`)**: 狂気の刃。HPと防御力が **半減（0.5倍）** する代わりに、攻撃力と速度が **2.5倍** に跳ね上がる。

#### **🌱 成長型 (`growth`) の変異**
- **∞ 無限 (`unlimited`)**: 成長の限界を超越し、通常の成長率が **1.5倍** になる上、育成が進んでも **ステータス上昇の減衰（収穫逓減）を完全に無視（最低1.0倍保証）** する。
- **👾 捕食 (`parasitic`)**: 通常のトレーニング効率が **0.1倍** に低下するが、戦闘で得られる経験値が **30倍**、勝利報酬が **3倍** になり、実戦を通じて飛躍的に成長する。

#### **💎 生命力 (`vitality`) の変異**
- **💀 不死 (`undead`)**: 死の概念の欠落。寿命が **5倍** に伸び、体力消費速度も **0.2倍** に低下。病気に **一切かからず（病気率0倍）**、さらに **敗北によるロストを無効化** する。
- **🔮 結晶化 (`crystalized`)**: 硬いが脆いガラスの盾。防御力が **2倍** になり、病気に **一切かからない（病気率0倍）** が、HPが **半減（0.5倍）** する。
- **🧪 合成獣 (`chimera`)**: 無尽蔵の体力。HPが **2.5倍** になるが、常に激しい拒絶反応により病気にかかる確率が **3倍** になる。

#### **💥 潜在能力 (`potential`) の変異**
- **🔥 超新星 (`supernova`)**: 個体値の振れ幅が最大（**0.5〜2.5倍**）となり、次世代への突然変異率を **5倍** に高める。
- **🌀 特異点 (`singularity`)**: 絶対不変の完成形。個体値のブレが一切なく（**固定1.0倍**）、両親のIVのうち最強の値を確実に継承する。子に必ず遺伝するが、**寿命の延長が不可** となる。
- **🚫 対禁忌 (`anti_taboo`)**: 禁忌を討つ者。相手の「禁忌深度」を看破し、攻撃力と防御力が向上（**+30% / 深度**）する。

#### **🧬 突然変異の継承ルール**
突然変異形質は、一度発現すると以下のルールに従って次世代へ引き継ぐことが可能です。
- **原則としての遺伝**: 突然変異は「特殊なアレル」として遺伝子型に組み込まれるため、メンデルの法則に従い、親から子へ（1/2の確率などで）引き継がれます。
- **例外：特異点（singularity）の劣化**: 唯一の例外として、「特異点」は次世代へ継承される際に形質を維持できず、強制的に「安定（stable）」へと劣化して遺伝します。
- **発生と継承の共存**: 交配において「既存の変異の継承」と「新しい変異の発生」は独立して判定されます。
- **強力な変異の固定 (Fixation)**: 「特異点」「対禁忌」「背反」などの強力な変異は、発現時に `MutationEngine` によって遺伝子座がホモ接合化（固定）されます。これにより、次世代へ確実に形質を継承することが可能です。
- **特異点（singularity）の不変性**: 以前の仕様とは異なり、現在の「特異点」は次世代へ継承される際も劣化せず、その絶対的な能力を維持し続けます。

#### **☯️ 特殊な変貌（血統の昇華）**
- **☯️ 背反 (`antinomy`)**: 禁忌を受け入れた反逆者。成長は鈍るが、禁忌による「不妊」や「寿命減衰」を克服し、呪われた血脈を次代へ繋ぐ架け橋となる。
通常の突然変異では発生しない形質

### 5.5 臨界点（伝承上の記述）
禁忌の歴史において、特定の血脈と特定の形質が交わった際、かつての世界の理を塗り替える絶対的な存在が生まれたと言い伝えられています。その存在は圧倒的な力を振るう代わりに、一切の繁殖手段と延命を拒むとされています。

## 6. 戦闘システム
### 6.1 決闘（対人戦）
他のユーザーの怒武者と 1vs1 で戦います。
- **リスク**: 敗北した個体は原則「死亡」となり、**データは永久にロスト**します。
- **完全同意制**:
    *   すべての挑戦に対して、防衛側は「受諾」または「拒否」を自由に選択可能です（ボクシング形式の廃止）。
    - **防衛側の選択権**: 防衛側が受諾した場合、防衛側も自分の戦わせる怒武者を選択します（複数所持時）。
- **プライバシー保護**: 他人の詳細ステータスは確認できません。

### 6.2 バトルロジック
- **ダメージ公式**: `(Atk^2 / (Atk + Def)) * 技威力 * 属性倍率 * 乱数`
- **クリティカル**: 倍率 **1.3倍**。
- **実況**: Discordメッセージによるリアルタイム実況（HPバー付き）。

## 7. 情緒的UI（ロマンチックUI）
効率や数値よりも「体験」を重視したデザインです。
- **等幅コードブロック**: 数値と項目が完璧に整列。
- **ANSIカラーコード**: 状態に応じた発色。
- **ビジュアルHPバー**: `[████░░░░]` による視覚的な直感性。
- **成長の抽象化**: 数値表示ではなく情緒的な記述で変化を伝えます。
- **プライバシー・隠蔽化**: 公開メッセージでは主要な数値が `???` で隠蔽されます。

## 8. 経済バランスとデフレ対策
- **時価アイテム（計画中）**: 延命アイテム等の価格をサーバー流通量に連動させインフレ抑制。
- **死亡リスク**: 敗北によるデータ消失（ポイント投入のリセット）により経済循環。

## 9. 技術的要件・永続化
- **ログ管理**: 生成・更新・死亡の全工程をロギング。
- **セッション復旧**: Bot再起動時に再計算・自動完結。
- **不正防止**: 個体値バリデーションによる整合性確保。
